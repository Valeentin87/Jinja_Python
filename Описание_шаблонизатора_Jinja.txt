официальный сайт по изучению работы шаблонизатора jinja https://jinja.palletsprojects.com/en/2.11.x/
1.для установки в терминале набираем команду pip install jinja
2.-------пример 1 ----------
from jinja2 import Template

name = 'Фёдор'
tm = Template('Привет {{name}}') # создали экземпляр класса Template
print(tm.render(name=name))  # применили метод render к переменной tm, позволяющий в шаблон вставить имя

метод render по сути действует как словарь в {{ }} указывается название ключа, а в скобках render(......) уже
присваивается ему значение

3. ------------синтаксис jinja -----------------
3.1 {% %} - спецификатор шаблона
3.2 {{ }} - конструкция для вставки любых выражений Python в шаблон
3.3 {#  #} - блок комментарий
3.4 {% raw %} .........{% endraw %} - экранирование(всё, что находится внутри будет изображено в шаблоне именно так)
3.5 {{ name | e }}     - экранирование отдельных символов в шаблонах
3.6 ---использование цикла for------
{% for <выражение> - %}
    <повторяемое выражение>
{% endfor %}
3.7 ----использование условного оператора if-------
{% if <условие> %}
    <действие, если условие истинно>
{% endif %}
